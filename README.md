# Campus_Yacu Website

##  Project Overview
The Campus Information Website is a static web application designed to provide students with easy access to important campus locations, student services, and announcements. The website focuses on clear content organization, structured layouts, and effective use of CSS for positioning and alignment.

This project demonstrates practical skills in HTML and CSS, particularly in building navigation systems, dropdown menus, service cards, dashboards, and sticky elements.

## Features
- **Navigation Bar with Dropdown Menus**
  - Easy access to locations, services, and announcements
- **Sticky Notice Bar**
  - Important information remains visible while scrolling
- **Announcements Section**
  - Displays recent campus updates and an important highlighted notice
- **Student Services Section**
  - Service cards presenting key student support services
- **Campus' location**
  - Grid-based layout showcasing important campus locations
- **Clean and Responsive Layout**
  - Uses Flexbox and CSS Grid for alignment and structure

---

## Technologies Used
- **HTML5** – Structure and content
- **CSS** – Styling, layout, positioning, and responsiveness
